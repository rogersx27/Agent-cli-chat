# CLI Chat Application

A simple CLI chat application built with Python, Typer, and Rich.

## Installation

Create and activate a virtual environment:

```bash
# Windows
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## Usage

Start the server:
```bash
python cli.py start-server
```

Connect as a client:
```bash
python cli.py connect <username>
```
